Detailed Description

template<size_t CAPACITY, typename ValueT>
class MatrixEntryArray< CAPACITY, ValueT >

Fixed capacity storage of an array of matrix index and associated value.

This implementation is minimal, only to serve the needs of various internal implementation details of linear algebra operations - it is not intended to be used as public API
