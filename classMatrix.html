Detailed Description

template<size_t ROWS, size_t COLS, typename ValueT = float>
class Matrix< ROWS, COLS, ValueT >

Class representing a ROWS x COLS dense matrix.

The row & column count(s) (ROWS, and COLS) are templated so that the required memory allocation of a given matrix is known at compile time, thus can be allocated on the stack.

ROWS and COLS define the shape of the matrix.

Template Parameters
ROWSnumber of rows in this matrix.
COLSnumber of columns in this matrix.
ValueTvalue type of the entries.
