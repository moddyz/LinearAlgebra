Detailed Description

Determinant computation.

Matrix inversion.

The determinant of a square matrix is computed by elimination, bringing A to U, then multiplying the pivots.

If the input matrix is singular, then the determinant is 0. If it is non-singular, then the determinant is non-zero.

The inverse matrix of A is the unique matrix such that

A * A^-1 = I

where I is the identity matrix.
