Detailed Description

Compile-time recursion for performing an unary or binary operation on every single entry of a set of Sequence objects.

Template function recursion and index sequence techniques are both used in the.

A Sequence object is defined to support indexing via operator[].

For example, to add the corresponding entries of matrices A and B, and store into matrix C:

linear::Matrix< 5, 5 > A, B;
linear::Matrix< 5, 5 > C = linear::SequenceBinaryOperation( std::plus< float >(), A, B );

Although matrices are traditionally indexed by two components (row, column), the Matrix::operator[] has been overloaded to perform a operations on every single entry

The terminating overload of the sequence * sequence MutableSequenceBinaryOperation.

The operational overload of a binary operation performed on two sequences.

This is a variation of SequenceBinaryOperation which cannot be used in a constexpr, because it modifies the memory of o_output. For example, this is used to implement the arithmetic assignment operators of the Matrix class.

Precondition
The shape of i_lhs, i_rhs, and o_output must be the same!

This overload code path is taken when Index is not equal the sequence count.
