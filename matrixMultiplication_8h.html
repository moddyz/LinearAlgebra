Detailed Description

Matrix mutiplication implementation details of compile-time supported matrix multiplication.

The entry point is linear::_MatrixMult, so read from bottom up.

An index sequence is generated with the same length as the column count of i_lhs (or, row count of i_rhs).

Forwards this inner product index sequence to _InnerProductIndexExpansion, for expansion.

The row and column indices are resolved, row-major-wise, into row and column indices.

Returns
the inner product.

Inner product computation of the RowIndex'th row of i_lhs and the ColumnIndex'th column of i_rhs.

Expands the index sequence into packed parameters and perform a sum of the product(s), for N number of entries:

lhs( Row, 0 ) * rhs( 0, Col ) + lhs( Row, 1 ) * rhs( 1, Col ) + ... + lhs( Row, N ) * rhs( N, Col )
Returns
the inner product.

Expands the index sequence into a packed parameters to construct MatrixProductT, via a fold expression.

Each parameter is computed as the _InnerProduct of a row of i_lhs and a column of i_rhs.

the matrix product.
