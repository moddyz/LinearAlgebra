Detailed Description

Convience iteration over a range of integers, with bi-directional support.

The integers in the range are inclusive of i_begin and exclusive of i_end.

Initialize this integer range.

If i_end > i_begin, the increment value is 1 for forwards iteration. If i_end < i_begin, the increment value is -1, for backwards iteration.

Note
If i_end == i_begin, the increment value is 0 - there will be no range to iterate over.
